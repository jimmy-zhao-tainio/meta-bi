# Business Data Vault SQL Demo

This demo is no longer an active SQL-generation demo.

`meta-datavault-business generate-metasql` is currently a stub, and the old DataVault-to-Sql projection path has been removed.

What remains here:

- `BusinessDataVault`
- `Implementation`
- `DataTypeConversion`

These workspaces are still useful as sanctioned input examples. The checked-in generated SQL-model output has been removed from the active surface.

If SQL deployment work resumes, it should start from sanctioned `SqlModel`, not from the removed projection code.
